jarjar

Description:

Repackages java library using jarjar.

Result:

Blob - jar containing repackages classes from "in" jar using provided rules.

Parameters:

String rules
jarjar rules, see Rules file format for details.
Blob in
jar containing classes to be repackaged

Examples:

Repackages classes in "jars/myJar.jar" file according to given rules.

repackaged.jar:
        file("jars/myJar.jar")
      | jarjar(rules="rule net.example.* org.repackaged-example.$@1");
