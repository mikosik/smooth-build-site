junit

Description:

Executes junit tests and fails if any of the test fail.

Result:

String - "SUCCESS" when all tests passed or "FAILURE" when some failed.

Parameters:

File* libs
collection of jar files containing test classes and all library classes needed for test to run. All classes which names end with Test will be run. Future version of smooth will provide fine-grained way to filter out tests to be run.

Examples:

Takes all files from "src" directory, compiles them and executes all tests among them. Note that in current releases there's no need to pass jar with junit classes. Junit binaries are embedded inside smooth-all.jar. This will probably change in future so it would be easy to update junit library without updating smooth version.

testJar: files("src") | javac | jar;
test: [ testJar ]  | junit;
