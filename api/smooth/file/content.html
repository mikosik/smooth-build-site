content

Description:

Converts File to Blob. In most cases you don't need to use this function conversion between File and Blob happens automatically. However there are cases when you want to force conversion to forbid automatic parameter assignment.

Result:

Blob containing content of give File.

Parameters:

File file
File to be converted.

Examples:

Creates array containing one Blob.


blobArray: [ content( file("src/LifeOfBrian.txt") ) ] ;
