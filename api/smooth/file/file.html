file

Description:

Reads single file from project filesystem.

Result:

File read from filesystem.

Parameters:

String path
Path to the file inside project file hierarchy. Referencing files outside project is not allowed. Slash ("/") should be used as separator no matter what your operating system is.

Examples:

Takes "src/LifeOfBrain.txt" file and stores it in "output" directory.


copyBook: file("src/LifeOfBrian.txt") | save("output");
