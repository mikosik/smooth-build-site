save

Description:

Saves single File or set of files (File*) in given directory in project filesystem. If directory is missing the it will be created.

Result:

Void

Parameters:

String dir
Path to destination directory within project filesystem. If directory is missing the it will be created. Referencing directories outside project is not allowed. Slash ("/") should be used as separator no matter what your operating system is.
File* files
Set of files to be stored in project filesystem
File file
File to be stored in project filesystem

Examples:

Takes "src/LifeOfBrain.txt" file and stores it in "output" directory.


copyBook: file("src/LifeOfBrian.txt") | save("output");
