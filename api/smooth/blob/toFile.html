toFile

Description:

Creates File from given path and content

Result:

File with given path and content.

Parameters:

String path
Path of returned File.
Blob content
Content of returned File.

Examples:

Creates file with "txt/my-file.txt" path and "This is file content" and zips it.


zipped-file: toFile("txt/my-file.txt", toBlob("This is file content")) | zip ;
