zip

Description:

Zips a set of files.

Result:

File named "output.zip".

Parameters:

File* files
set of files to be compressed (ziped)
String output
Path of output file. Defaults to "output.zip".
File manifest
file to be used as manifest

Examples:

Takes all files from "src" directory, zips them into "output.zip" and stores in "output_dir" directory.


files.zip: files("src") | zip ;
