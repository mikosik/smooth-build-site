unzip

Description:

Unzips files from given zip file.

Result:

File* - set of files from given zip file.

Parameters:

File file
zip file to be uncompressed

Examples:

Takes "zips/myZip.zip" file and uncompresses all files from it to "output" directory.


fileFromZip: file("zips/myZip.zip") | unzip | save("output/");
