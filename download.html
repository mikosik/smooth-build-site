Downloading and Installation


Download the latest Smooth binary from smooth-0.1.0.zip and unzip it. It contains one "smooth" directory.

Download Java Development Kit version 7 (or newer) and install it on your system. It is needed as Smooth is implemented in Java.

Smooth can be run by one of script files unzipped from smooth.zip.
  • "smooth/smooth" script on Linux systems
  • "smooth/smooth.bat" file on Windows system
You may add unzipped "smooth" directory to your system PATH to make Smooth runnable from any directory.
